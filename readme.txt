== Description ==
A plugin which converts currencies using API.

== How to Use ==

After Installing paste ['CurrencyCalculator'] it will display conversion form on the page.

== Requirements ==
It can run on any wordpress version.

== Frequently Asked Questions ==

= Can I use my API keys ? =
  Yes you can use user API keys.

= Can I change API URL ?
  Yes you can pass your API URL.
